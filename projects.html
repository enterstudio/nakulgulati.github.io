Location Based Customization App

An Android app that customizes phones settings viz. Wi-Fi, Bluetooth and data based on user's location. The user can group settings which are then associated with an area. The area is set using Android's Geofence API. Currently areas can be approximated into circular areas using radius as a parameter.
